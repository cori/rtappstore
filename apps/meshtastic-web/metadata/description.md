# Meshtastic Web

The official web interface for [Meshtastic](https://meshtastic.org) mesh networking devices.

## Features

- **Device Configuration** — Configure radio settings, channels, and modules directly from your browser
- **Messaging** — Send and receive messages across the mesh network
- **Map View** — Visualize node positions on an interactive map
- **Node Management** — Monitor connected nodes, signal quality, and battery levels
- **Channel Management** — Create and manage encrypted communication channels
- **Multi-Transport Support** — Connect via HTTP, Bluetooth, or Serial

## Usage

After installation, open the web interface and connect to your Meshtastic device. The device must be reachable on the same network (for HTTP transport) or via Bluetooth/Serial from the host machine.
